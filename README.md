8# AI Resume Analyzer

## About the Project

I built this project to help students and job seekers compare their resumes with a job description and identify missing skills.

The application allows users to upload a resume in PDF format, extracts the text from the resume, and compares it against the skills mentioned in a job description.

The goal is to help users understand where their resume matches the requirements and which skills they may need to improve.

## Features

* Upload resume in PDF format
* Extract text from resume automatically
* Compare resume skills with job requirements
* Calculate a match percentage
* Display matching and missing skills
* Simple and user-friendly interface

## Technologies Used

* Python
* Streamlit
* PDFPlumber
* Git & GitHub
